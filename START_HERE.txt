TROUBLESHOOTING
===============

Problem: "Python not found"
Solution: Install from https://www.python.org
         During install, check "Add Python to PATH"
         Restart command prompt and try again

Problem: "Permission denied"
Solution: Close HTML files in editors
         Run command prompt as Administrator
         Check file properties (remove read-only)

Problem: "File not found"
Solution: Make sure script is in correct folder:
         c:\xampp\htdocs\Webbuilders Projects\A2Z\A2Z

Problem: "Some updates failed"
Solution: See FOOTER_UPDATE_INSTRUCTIONS.md
         Check error message for specific file
         Verify file isn't corrupted

For more details: See FOOTER_UPDATE_INSTRUCTIONS.md
